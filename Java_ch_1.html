Java Syntax and Output

अध्याय 1 (Chapter 1)

Java क्या है?

Java एक लोकप्रिय और शक्तिशाली प्रोग्रामिंग भाषा है, जिसे वर्ष 1995 में बनाया गया था।

यह Oracle कंपनी के स्वामित्व में है, और आज 3 बिलियन से अधिक उपकरण Java पर चलते हैं।

Java का उपयोग कहाँ होता है?

  • मोबाइल एप्लिकेशन (विशेष रूप से एंड्रॉइड ऐप्स)
  • डेस्कटॉप एप्लिकेशन
  • वेब एप्लिकेशन
  • वेब सर्वर और एप्लिकेशन सर्वर
  • गेम्स
  • डेटाबेस कनेक्शन
  • और भी बहुत कुछ!

Java का उपयोग क्यों करें?

  • Java अलग-अलग प्लेटफॉर्म (Windows, Mac, Linux, Raspberry Pi आदि) पर काम करता है।
  • यह दुनिया की सबसे लोकप्रिय प्रोग्रामिंग भाषाओं में से एक है।
  • वर्तमान नौकरी बाज़ार में इसकी बहुत अधिक मांग है।
  • यह सीखने में आसान और उपयोग में सरल है।
  • यह ओपन-सोर्स और मुफ्त है।
  • यह सुरक्षित (Secure), तेज़ (Fast) और शक्तिशाली (Powerful) है।
  • इसकी एक विशाल समुदाय सहायता है (दसियों लाख डेवलपर्स)।
  • Java एक ऑब्जेक्ट ओरिएंटेड भाषा है जो प्रोग्राम्स को स्पष्ट संरचना देती है और कोड के पुन: उपयोग की अनुमति देती है, जिससे विकास की लागत कम होती है।
  • चूँकि Java, C++ और C# के काफ़ी करीब है, इसलिए प्रोग्रामर आसानी से इन भाषाओं के बीच स्विच कर सकते हैं।

Java में Naming Styles (केस कन्वेंशन)

जावा में, हम अलग-अलग चीज़ों (Elements) को नाम देने के लिए कुछ खास नियम (Rules) मानते हैं। इन्हें Case Conventions कहते हैं। इन्हें फॉलो करने से हमारा कोड पढ़ने में आसान (Easy to read) और साफ़-सुथरा (Clean) लगता है।

1. CamelCase (कैमल केस)

पहचान: नाम हमेशा एक छोटे अक्षर (lowercase) से शुरू होता है, और उसके बाद आने वाले हर नए शब्द का पहला अक्षर बड़ा (Capital) होता है।

उदाहरण (Examples): firstName, calculateSalary

उपयोग (Usage) और उदाहरण:

  • वेरिएबल (Variable): int maxAdultUserOfYear2025;
  • मेथड (Method): public void calculateSalary() { ... }
  • ऑब्जेक्ट (Object): Employee empObject = new Employee();

2. PascalCase (पास्कल केस)

पहचान: नाम के हर शब्द का पहला अक्षर बड़ा (Capital) होता है।

उदाहरण (Examples): String, Employee, MaxAdultUser

उपयोग (Usage) और उदाहरण:

  • क्लास (Class): public class Employee { ... }
  • इंटरफ़ेस (Interface): interface Clickable { ... }, interface Runnable { ... }
  • एक्सेप्शन (Exception): class FileNotFoundException { ... }

3. SCREAMING_SNAKE_CASE (स्क्रीमिंग स्नेक केस)

पहचान: सभी अक्षर बड़े (Uppercase) होते हैं, और शब्दों के बीच में अंडरस्कोर (_) लगाया जाता है।

उदाहरण (Examples): MAX_VALUE, MAX_USER_COUNT

उपयोग (Usage) और उदाहरण:

  • कांस्टेंट (Constants): public static final int MAX_VALUE = 100;
  • फाइनल वेरिएबल: static final double PI = 3.14;

4. lowercase dot notation (लोअरकेस डॉट नोटेशन)

पहचान: सभी अक्षर छोटे (lowercase) होते हैं, और शब्दों को डॉट (.) से अलग किया जाता है।

उदाहरण (Examples): com.google.app, com.q.c

उपयोग (Usage) और उदाहरण:

  • पैकेज (Package): package com.mycompany.project;
  • रिवर्स डोमेन नाम: package org.apache.commons;

केस कन्वेंशन की समरी (Summary of Case Conventions)

कन्वेंशन (Convention) नाम लिखने का तरीका (Style Example) मुख्य उपयोग (Used for)
CamelCase myVariable calculateTax() empObject वेरिएबल, मेथड्स, ऑब्जेक्ट
PascalCase MyClass Runnable IOException क्लास, इंटरफ़ेस, एक्सेप्शन
SCREAMING_SNAKE_CASE MAX_USERS DEFAULT_SIZE कांस्टेंट (static final variables)
lowercase dot notation com.app.name java.util पैकेज नाम


जावा कोड लिखना और चलाना (Writing and Running Java Code)

1. फाइल बनाना (Creating a File)

Linux Terminal या Termux में nano कमांड का उपयोग करके किसी भी नाम से .java (small letter) extension की file create करें। File name PascalCase में रखें।

$$cmd \rightarrow nano \ MyFirstCode.java \rightarrow Enter$$

2. कोड लिखना (Writing the Code)

अब जो window open होगी उसमें नीचे का कोड लिखें:

public class MyFirstCode {
public static void main(String[] args) {
System.out.println("Hello World!");
}
}

3. सेव और बाहर निकलना (Saving and Exiting)

अब Ctrl + O दबाकर फ़ाइल को save करने की प्रक्रिया शुरू करें,
फिर Enter दबाकर फ़ाइल को वास्तव में save करें।
इस window (विंडो) से बाहर जाने के लिए Ctrl + X दबाएँ।

4. कंपाइल करना (Compiling)

Source Code को compile करने के लिए:

$$cmd \rightarrow javac \ MyFirstCode.java \rightarrow Enter$$

अगर आपके source code में कोई error नहीं होगी तो cursor अगली लाइन में आ जाएगा। इससे MyFirstCode.class (bytecode file) बन जाएगी।

5. फाइल्स देखना (Viewing Files)

अब अगर source file और class file देखना चाहते हैं तो यह Command रन करें

$$cmd \rightarrow ls \ \rightarrow Enter$$

आपको MyFirstCode.java और MyFirstCode.class दिखाई। अब इस class file यानी bytecode file या compiled java file को run यानी Execute करने के लिए ये नीचे के step की cmd यूज़ करें।

6. रन करना (Running)

Compiled Java file (class file) को run (Execute) करने के लिए:

$$cmd \rightarrow java \ MyFirstCode \rightarrow Enter$$

आपको आउटपुट मिलेगा: Hello World!

जावा कोड की व्याख्या (Explanation of Java Code)

इस जावा कोड में सबकुछ lower case में रहेगा, बस main method की String[] में S, और System में S को छोड़कर।

आप सिर्फ दो चीज़ें अपनी मर्ज़ी से लिख सकते हैं: Class का नाम (जो file का नाम होता है और PascalCase में होता है) और " " के बीच का Text (जो screen पर दिखाना चाहते हैं)।

The main Method

main() method प्रत्येक जावा कोड में होनी चाहिए। यह वह जगह है जहाँ से जावा प्रोग्राम run होना start होता है।

public static void main(String[] args)

Curly Braces { }

  • curly braces { } कोड के ब्लॉक की शुरुआत और अंत को बताते हैं।

System.out.println() के भाग

  • System: यह एक built-in java class है।
  • out: System का एक member है जो output को दिखाता है।
  • println(): यह एक method है, जो print Line का short है।

हमें प्रत्येक जावा statement के अन्त में semicolon (;) लगाना होगा।

जावा में Output Statement के प्रकार

1. print(): प्रिंट के बाद cursor उसी लाइन में रहता है।

2. println(): प्रिंट के बाद नई लाइन ऐड करता है, यानि cursor अगली लाइन में चला जाता है।

\n (New Line Escape Sequence): न्यूलाइन (Text) के अन्दर नई लाइन के लिए उपयोग होता है।

कोड उदाहरण (Code Example) 1.1

public class OutputExample1 {
public static void main(String[] args) {
System.out.print("Hello ");
System.out.println("World!");
System.out.println("Line\nLine2");
}
}

आउटपुट (Output)

Hello World!
Line
Line2

Java Escape Sequences (एस्केप सीक्वेंस)

Java में characters को escape करने के लिए \ का प्रयोग करना पड़ता है।

कुछ common Java escape sequences:

  • \\      backslash
  • \"      double quote (String के अन्दर)
  • \'      single quote
  • \uXXXX    Unicode escape
  • \n      newline
  • \r      carriage return
  • \t      tab
  • \b      backspace

🧩 उदाहरण प्रश्न 1.2:

एक Java प्रोग्राम लिखिए जो स्क्रीन पर नीचे दिया गया वाक्य बिल्कुल वैसे ही प्रिंट करे:

He said, "Can you do it?"

आउटपुट बिल्कुल इसी तरह दिखना चाहिए, जिसमें डबल कोट्स (") और प्रश्नवाचक चिन्ह (?) भी शामिल हों।

💡 संकेत: स्ट्रिंग के अंदर डबल कोट्स प्रिंट करने के लिए एस्केप सीक्वेंस (\") का उपयोग करें।

✅ Solution (Java Code):

public class Main {
    public static void main(String[] args) {
        System.out.println("He said, \"Can you do it?\"");
    }
}

🖥️ Output:

He said, "Can you do it?"

💻 Java में Escape Characters का अभ्यास - 1.1

नीचे दिए गए प्रश्नों में आपको print, println और सभी escape characters का उपयोग करना है:

🧩 प्रश्न 1: New Line और Carriage Return

प्रश्न: एक Java प्रोग्राम लिखो जो नीचे दिया गया आउटपुट दे:
Hello
Welcome to Java

संकेत:
"Hello" के बाद नई लाइन \n लगाओ।
"World" प्रिंट करो लेकिन फिर "Welcome to Java" का उपयोग करके इसे overwrite कर दो।
यानी \r का उपयोग कर "World" की जगह "Welcome to Java" दिखाओ।

✅ समाधान:

public class Q1_NewLineCarriage {
    public static void main(String[] args) {
        System.out.print("Hello\n");
        System.out.print("World\rWelcome to Java\n");
    }
}

💻 आउटपुट:

Hello
Welcome to Java

🧩 प्रश्न 2: Tab और Backspace

प्रश्न: एक Java प्रोग्राम लिखो जो नीचे दिया गया आउटपुट दे:
Name: Ram
City: Noida
RollNo: 01

संकेत:
- \t का उपयोग spacing के लिए करो ताकि कॉलम अच्छे से align हों।
- \b का उपयोग करके "RollNo" में गलत अंक को हटाओ और सही संख्या दिखाओ।
Hint: '1\b01' का उपयोग करके output में "01" प्राप्त किया जा सकता है।

✅ समाधान:

public class Q2_TabBackspace {
    public static void main(String[] args) {
        System.out.println("Name:\tRam");
        System.out.println("City:\tNoida");
        System.out.print("RollNo:\t1\b01\n");
    }
}

💻 आउटपुट:

Name:    Ram
City:    Noida
RollNo:  01

संकेत: \' और \" दोनों का प्रयोग करो।

public class Q4_Quotes {

🧩 प्रश्न 4: Backslash और Combination Escape Characters

प्रश्न: नीचे जैसा आउटपुट देने वाला प्रोग्राम लिखो:
C:\Program Files\Java\bin
"Java\tCode\nPractice"
संकेत: \\ बैकस्लैश प्रिंट करता है और बाकी escape characters को टेक्स्ट के रूप में दिखाओ।

✅ समाधान:

public class Q5_BackslashCombo {

        System.out.println("C:\\Program Files\\Java\\bin");
        System.out.println("\"Java\\tCode\\nPractice\"");

}

Print Numbers संख्याएँ प्रिंट करना

आप संख्याएँ प्रिंट करने के लिए println() मेथड का भी उपयोग कर सकते हैं।

हालाँकि, टेक्स्ट की तरह हम संख्याओं को डबल कोट्स ("") में नहीं डालते:

System.out.println(3);

358

आप println() मेथड के अंदर mathematical calculations (गणितीय गणना) भी कर सकते हैं:

उदाहरण:

System.out.println(3 + 3);
System.out.println(2 * 5);

Output:

6
10

Java Comments जावा कमेंट्स

Comments का उपयोग Java कोड को समझाने और उसे more readable(अधिक पठनीय) बनाने के लिए किया जा सकता है।

यह alternative code को test करते समय execution को रोकने के लिए भी इस्तेमाल किया जा सकता है।

Single-line Comments एक लाइन के कमेंट्स

Single-line comments दो forward slashes (//) से शुरू होते हैं।

// और लाइन के अंत के बीच का कोई भी text Java द्वारा ignore किया जाता है (execute नहीं होगा)।

उदाहरण:

// This is a comment
System.out.println("Hello World");

Output आउटपुट:


Hello World

लाइन के अंत में भी single-line comment इस्तेमाल किया जा सकता है:

उदाहरण:

System.out.println("Hello World"); // This is a comment

Output आउटपुट:

Hello World

Java Multi-line Comments मल्टी-लाइन कमेंट्स

Multi-line comments /* से शुरू होते हैं और */ से खत्म होते हैं।

/* और */ के बीच का कोई भी text Java द्वारा ignore किया जाएगा।

उदाहरण:

/* The code below will print the words Hello World
   to the screen, and it is amazing */
System.out.println("Hello World");

Output आउटपुट:

Hello World

Single or Multi-line comments? एक या दोनों?

आप जिसको चाहो इस्तेमाल कर सकते हैं। आमतौर पर, छोटे comments के लिए // और लंबे comments के लिए /* */ का उपयोग किया जाता है।

🎯 प्रश्न 5: व्यक्तिगत बजट कैलकुलेटर

वास्तविक जीवन का परिदृश्य:

आप अपने मासिक खर्चों को ट्रैक करना चाहते हैं और यह देखना चाहते हैं कि सभी खर्चों के बाद आपके पास कितना पैसा बचता है।

समस्या:

एक प्रोग्राम बनाएं जो आपकी मासिक आय और विभिन्न खर्चों को ले, फिर गणना करे कि कितना पैसा बचा है।

💡 संकेत:

  • System.out.print() का उपयोग input values के लिए करें
  • आय और विभिन्न खर्चों को store करने के लिए variables का उपयोग करें
  • कुल खर्चों को आय से घटाकर शेष राशि प्राप्त करें
  • बेहतर formatting के लिए \n और \t जैसे escape sequences का उपयोग करें

✅ समाधान कोड:

public class BudgetCalculator { public static void main(String[] args) { // वेरिएबल्स इनिशियलाइज़ करें double income = 25000.0; double rent = 8000.0; double food = 5000.0; double transport = 2000.0; double entertainment = 3000.0; // कुल खर्च और शेष राशि की गणना करें double totalExpenses = rent + food + transport + entertainment; double remaining = income - totalExpenses; // परिणाम दिखाएँ System.out.println("=== व्यक्तिगत बजट कैलकुलेटर ==="); System.out.println("मासिक आय:\t\t₹" + income); System.out.println("\n--- खर्च ---"); System.out.println("किराया:\t\t\t₹" + rent); System.out.println("भोजन:\t\t\t₹" + food); System.out.println("यातायात:\t\t₹" + transport); System.out.println("मनोरंजन:\t\t₹" + entertainment); System.out.println("-------------------------------"); System.out.println("कुल खर्च:\t\t₹" + totalExpenses); System.out.println("\nशेष राशि:\t\t₹" + remaining); // शेष राशि के आधार पर अतिरिक्त संदेश if (remaining > 0) { System.out.println("\nबहुत अच्छा! इस महीने आपके पास बचत है! 💰"); } else { System.out.println("\nसावधान! आप कमाई से ज्यादा खर्च कर रहे हैं! ⚠️"); } } }

🖥️ अपेक्षित आउटपुट:

=== व्यक्तिगत बजट कैलकुलेटर === मासिक आय: ₹25000.0 --- खर्च --- किराया: ₹8000.0 भोजन: ₹5000.0 यातायात: ₹2000.0 मनोरंजन: ₹3000.0 ------------------------------- कुल खर्च: ₹18000.0 शेष राशि: ₹7000.0 बहुत अच्छा! इस महीने आपके पास बचत है! 💰

🎯 प्रश्न 6: छात्र ग्रेड कार्ड जनरेटर

वास्तविक जीवन का परिदृश्य:

एक प्रोग्राम बनाएं जो विभिन्न विषयों में अंकों के साथ छात्र का ग्रेड कार्ड दिखाए और प्रतिशत की गणना करे।

समस्या:

एक प्रोग्राम डिज़ाइन करें जो छात्र के विवरण, विषयों के अंक, कुल अंक और प्रतिशत के साथ एक फॉर्मेटेड ग्रेड कार्ड दिखाए।

💡 संकेत:

  • फॉर्मेटिंग के लिए कई println() स्टेटमेंट्स का उपयोग करें
  • अलाइनमेंट के लिए \t escape sequence का उपयोग करें
  • कुल अंक और प्रतिशत की गणना करें
  • दिखने में आकर्षक रिपोर्ट कार्ड लेआउट बनाएं

✅ समाधान कोड:

public class GradeCardGenerator { public static void main(String[] args) { // छात्र की जानकारी String studentName = "राम यादव"; String rollNumber = "2024001"; String className = "X - A"; // विषयों के अंक int english = 85; int hindi = 78; int mathematics = 92; int science = 88; int socialScience = 81; // गणनाएँ int totalMarks = english + hindi + mathematics + science + socialScience; double percentage = (totalMarks / 500.0) * 100; // ग्रेड कार्ड दिखाएँ System.out.println("========================================"); System.out.println(" छात्र ग्रेड कार्ड "); System.out.println("========================================"); System.out.println("नाम:\t\t" + studentName); System.out.println("रोल नंबर:\t" + rollNumber); System.out.println("कक्षा:\t\t" + className); System.out.println("----------------------------------------"); System.out.println("विषय\t\t\tअंक"); System.out.println("----------------------------------------"); System.out.println("अंग्रेजी\t\t" + english + "/100"); System.out.println("हिंदी\t\t\t" + hindi + "/100"); System.out.println("गणित\t\t\t" + mathematics + "/100"); System.out.println("विज्ञान\t\t\t" + science + "/100"); System.out.println("सामाजिक विज्ञान\t" + socialScience + "/100"); System.out.println("----------------------------------------"); System.out.println("कुल\t\t\t" + totalMarks + "/500"); System.out.println("प्रतिशत\t\t\t" + percentage + "%"); System.out.println("========================================"); // प्रदर्शन संदेश if (percentage >= 90) { System.out.println("उत्कृष्ट प्रदर्शन! 🎉"); } else if (percentage >= 75) { System.out.println("बहुत अच्छा प्रदर्शन! 👍"); } else if (percentage >= 60) { System.out.println("अच्छा प्रदर्शन! 👏"); } else { System.out.println("मेहनत जारी रखें! 💪"); } } }

🖥️ अपेक्षित आउटपुट:

======================================== छात्र ग्रेड कार्ड ======================================== नाम: राम यादव रोल नंबर: 2024001 कक्षा: X - A ---------------------------------------- विषय अंक ---------------------------------------- अंग्रेजी 85/100 हिंदी 78/100 गणित 92/100 विज्ञान 88/100 सामाजिक विज्ञान 81/100 ---------------------------------------- कुल 424/500 प्रतिशत 84.8% ======================================== बहुत अच्छा प्रदर्शन! 👍

🧩 प्रश्न 7 :

एक Java प्रोग्राम लिखिए जो 0 से 10,000,000 तक गिनती करे और हर संख्या के लिए स्क्रीन पर प्रिंट करे कि कितने प्रतिशत प्रगति हो चुकी है। प्रोग्राम का आउटपुट इस फॉर्मेट में होना चाहिए:

progress 0% 0
progress 0% 1
progress 0% 2
...
progress 100% 10000000

ध्यान दें कि:

  • आपको for loop का उपयोग करना है।
  • प्रतिशत प्रगति की गणना इस प्रकार करनी है: (currentNumber * 100) / totalNumber
  • हर लाइन में "progress percentage% currentNumber" फॉर्मेट होना चाहिए।
  • कुल संख्या 10,000,000 है।

संकेत:

आपको दो int वेरिएबल की जरूरत होगी: एक वर्तमान संख्या के लिए और एक टोटल संख्या के लिए।

💡 समाधान (Solution):

public class Main {
    public static void main(String[] args) {
        int j = 0;
        int a = 10000000;

        for(int i = 0; i <= a; i++) {
            j = i;
            System.out.println("progress " + ((j * 100) / a) + "% " + i);
        }
    }
}

📌 अपेक्षित आउटपुट (Expected Output):

progress 0% 0
progress 0% 1
progress 0% 2
...
progress 100% 10000000

नोट: आउटपुट में "..." केवल बीच के संख्याओं को दिखाने के लिए है। वास्तविक प्रोग्राम सभी संख्याओं को 0 से 10,000,000 तक प्रिंट करेगा।
